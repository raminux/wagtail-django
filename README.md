# Wagtail Django Tutorial

## How to install
```bash
$> mkdir wagtail-django
$> cd wagtail-djago
$> pipenv shell
$> pip install wagtail
$> pip freeze | grep wagtail
$> wagtail start mysite
$> mv mysite site
$> cp -r site/* .
$> rm -rf site
$> pip install -r requirements.txt
$> python manage.py migrate
$> python manage.py createsuperuser
$> python manage.py runserver
```
